# 智能家居未来生活

AI 生成的网页项目。

## 文件结构

```
├── index.html          # 主页面
├── css/
│   └── styles.css      # 样式文件（Tailwind CSS 编译后）
├── js/
│   └── main.js         # 交互脚本
└── README.md           # 本文件
```

## 使用方法

直接在浏览器中打开 `index.html` 即可查看网页。

如需部署到服务器，将整个文件夹上传到 Web 服务器的根目录即可。

## 技术栈

- HTML5
- Tailwind CSS（已编译为静态 CSS）
- Lucide Icons
